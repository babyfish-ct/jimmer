## 和ORM的互动

动态性是Jimmer对象的内禀特征，具有普适性，ORM框架本身和开发人员双方都可以轻松构建动态对象给对方使用。

-   Jimmer创建动态对象，返回给开发人员

    即，查询任意形状的数据结构。此功能叫[对象抓取器](../query/object-fetcher)

-   开发人员创建动态对象，传递给Jimmer

    即，保存任意形状的数据结构。此功能叫[保存指令](../mutation/save-command)
